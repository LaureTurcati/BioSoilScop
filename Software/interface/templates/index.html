Mode d'emploi

Suivez ces étapes pour démarrer en toute simplicité.

1. Date

La date est importante pour le bon fonctionnement du CamTrap. Elle permet de dater les photos et vidéos prises par l'appareil.

⚠ Pensez à vérifier régulièrement l'heure du CamTrap.

2. Configurations

Accédez aux configurations pour régler :

  • Les paramètres de la caméra (qualité d'image, résolution).
  • La détection de mouvement (sensibilité).
  • La luminosité en fonction de l'environnement.

Ces paramètres influencent directement la qualité des enregistrements.

3. Création de session

Créez des sessions pour organiser vos observations. Lors de la création, configurez :

  • Nom de la session : Identifiez chaque session avec un nom unique.
  • Mode d’enregistrement : Continu ou basé sur des plages horaires.
  • Métadonnées : Ajoutez des informations importantes comme le nom du site, la localisation ou des détails spécifiques à la zone observée.

Le nom de la session est importante pour les captures.

4. Liste des sessions

Gérez vos différentes sessions :

  • Démarrez ou arrêtez une session.
  • Supprimez une session

⚠ Attention, la suppression d'une session est définitive. Elle entraine la suppression du dossier (photos, logs et les métadonnée de la session)

5. Drive

Le CamTrap offre un accès à un espace de stockage :

  1. Lancez l'outil Filebrowser.
  2. Ouvrez-le dans un nouvel onglet pour accéder aux dossiers.

Structure des dossiers :

  • session1/
    • session1.json
    • logs.json
    • images/
  • session2/
    • session2.json

Dans le schéma, session1 est une session qui a déjà été lancée, alors que session2 a juste été créée.

6. Stockage

Vous pouvez voir l'espace disque restant sur le CamTrap.

7. Mode de Connexion

Changez le mode de connexion selon vos besoins :

  • Mode Point d’Accès : Si aucun réseau wifi n'est disponible.
  • Mode Wi-Fi :

Assurez-vous que le CamTrap est connecté au même réseau que votre ordinateur.
